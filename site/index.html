Generating a Go API

In your project root folder create a dbcore.yml: 

          
          project: 
          database:
            dialect: postgres
            host: $host-or-ip # Omit if localhost
            port: $port # Omit if 5432
            database: $database-name
            username: $username
            password: $password
            parameters: ?sslmode=disable # For development, for production omit

          api:
            template: go
            outDir: go
            version: $version-prefix # e.g. "v1", for URLs
	    address: :8080
            extra:
              repo: $your-repo

Clone the repo and run dotnet run $path-to-your-project within the repo root.

Then go to your project directory and run go run cmd/main.go to start the server.
